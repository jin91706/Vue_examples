# MPD2 Vue 3 Template with TailwindCSS

## Requirements
Latest NPM
Latest NODE JS

## Project Setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles for Production
```
npm run build
```
### Alternatively. Run using Docker
```
// This command runs 'npm install & npm run dev'
docker-compose up -d
// -d option is for detached mode
```
